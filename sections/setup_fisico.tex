\section{Introducción}

El propósito del presente informe es entregar una guía de apoyo para comprender como se pudo realizar la modificación del montaje del motor de corriente continua. Anteriormente este contaba solamente con un motor de corriente continua y la única variable medida era la velocidad rotacional del eje del motor. Dada la necesidad de implementar un sistema de lazos anidados y además de aplicar una carga mecánica que ejerza torque al motor eléctrico, se implementó un sistema de medición de corriente para poder comunicar la corriente al programa que realiza el control, se acopló mecánicamente un generador eléctrico, y, además, se implementó un módulo de carga electrónica que permite ser controlado desde el computador para consumir mas o menos corriente según lo que se desee.\\

En las siguientes secciones se deescribirá en detalle como se conforma el sistema completo, tanto a nivel físico como computacional, y también, como se implementaron todas las adiciones.\\
